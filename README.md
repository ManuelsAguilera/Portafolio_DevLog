# ManuDevLog

Devlog personal donde documento lo que construyo mientras lo construyo — decisiones técnicas, errores, descubrimientos.

**Stack:** Next.js 15 (App Router) + Firebase (Firestore, Auth, Cloud Functions)

## Quick start

```bash
npm install
npm run dev
```
